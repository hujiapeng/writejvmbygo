1. 创建HelloWorld.java文件，并写代码如下<br/>
	<pre>
	public class HelloWorld{
        public static void main(String[] args){
            System.out.println("Hello World");
        }
    }
	</pre>

2. 编译文件：javac HelloWorld.java<br/>
   执行文件：java HelloWorld<br/>
